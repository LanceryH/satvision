## Summary
Satvision is a project made with python (Model/Control) and javascript (View)

The main objectifs:
- [x] 3D position
- [x] Groundtrack
- [ ] Stages mass optimizer
- [ ] Nozzle optimizer

> [!NOTE]
> Stages mass and nozzle optimizer is in progress

## Requirements

```
pip install numpy
pip install PyQt5
pip install PyQtWebEngine
```
