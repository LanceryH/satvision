## Summary
Satvision is a project made with python and javascript based on a MVC model.

The main objectifs:
- [x] 3D position
- [x] Groundtrack
- [ ] Stages mass optimizer
- [ ] Nozle optimizer

## Requirements
```bash
pip install numpy
pip install PyQt5
pip install PyQtWebEngine
```
